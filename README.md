# Cyrahespinosa Portfolio Website

A personal portfolio website showcasing my skills, projects, and contact info. Built with HTML, CSS, and JavaScript, featuring a sticky navbar, parallax video, and fully responsive design.

---

## Features

- Sticky navigation bar
- Parallax video in hero section
- Responsive layout for desktop, tablet, and mobile
- Skills section without bullets, centered alignment
- Projects section with grid layout
- Contact section with working buttons and social links
